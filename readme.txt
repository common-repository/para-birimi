Convert 19 different currencies. / 19 ülkenin para birimini çevirir.

== Description ==

Bu eklenti 19 farklı para birimini dönüştürür. Para birimleri için ShortCode oluşturur. Seçilen para birimleri için, bir para birimi tablosu Widget'ı oluşturur. Woocommerce kullanıcılarının ürün sayfalarına anlık döviz çevirici ekler. Sepet ve ödeme sayfalarında toplam tutarı 16 farklı para birimine dönüştürebilir.

PLUS VER : 
https://codecanyon.net/item/19-different-currencies-instant-converter-for-woocommerce/19466196?ref=Ozibal

DEMO :
http://dev.4gendesign.com/product/tcmbconverter_demo/

Özellikler:

* Turkiye Cumhuriyeti Merkez Bankası (TCMB) günlük doviz kurlarını gösterir.
* 19 değişik para birimi için çevrim yapabilir. (TRY=>TÜRK LİRASI, USD=>AMERİKAN DOLARI, AUD=>AVUSTRALYA DOLARI, DKK=>DANİMARKA KRONU, EUR=>EURO, GBP=>İNGİLİZ STERLİNİ, CHF=>İSVİÇRE FRANGI, SEK=>İSVEÇ KRONU, CAD=>KANADA DOLARI, KWD=>KUVEYT DİNARI, NOK=>NORVEÇ KRONU, SAR=>SUUDİ ARABİSTAN RİYALİ, JPY=>JAPON YENİ, BGN=>BULGAR LEVASI, RON=>RUMEN LEYİ, RUB=>RUS RUBLESİ, IRR=>İRAN RİYALİ, CNY=>ÇİN YUANI, PKR=>PAKİSTAN RUPİSİ).
* Tamamen CSS ile dizayn edilmiştir. Özelleştirebilirsiniz.
* ShortCode ve Widget.

PLUS VER:

* Woocommerce ürün sayfalarında, sepet sayfasında ve ödeme sayfasında plugin i aktif hale getirebilirsiniz.
* Mağaza kurunu otomatik tespit eder ve seçtiğiniz kur için anlık çevrim yapar.
* Ürün sayfalarında tüm fiyat tiplerinde çevrim yapar. (Tekli fiyat, indirimli fiyat veya fiyat aralığı)

Veri Kaynağı:

* Günlük kur verileri, Türkiye Cumhuriyeti Merkez Bankası sayfasından çekilmektedir.
* Veri kaynağı : http://www.tcmb.gov.tr/kurlar/today.xml .
* Verinin İşlenmesi : 1 USD = X TRY ==> X TRY = ? EUR.

Not: Bu işlem ufak küsarat farklılıkları oluşturmaktadır.


== Installation ==


- Eklenti klasörünü /wp-content/plugins/ klasörüne kopyalayın.
- Eklentiler menüsünden eklentiyi aktifleştirin.
- "Para Birimi" menüsünde eklentinizi özelleştirin.
